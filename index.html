Agentes de IA no Telegram.
Sem Docker. Sem complicação.

SMB Claw é um framework ultra leve para rodar múltiplos bots de IA no Telegram numa única VPS. Um script e você já tem um agente funcionando — com memória, ferramentas e personalidade própria.

Começar agora Ver no GitHub

Por que o SMB Claw existe? Depois de semanas passando raiva com o OpenClaw — que às vezes funciona, às vezes não, você pede uma coisa e ele faz outra — decidimos criar nossa própria alternativa com uma proposta diferente: extremamente simples, eficiente e confiável.

Sem precisar lidar com infraestrutura complexa como Docker ou Kubernetes. Basicamente um script e você já tem um agente de IA rodando no Telegram — com memória persistente, ferramentas modulares e personalidade própria. Do jeito que deveria ser desde o começo.

Recursos

Um framework completo sem a complexidade que ninguém pediu.

Ultra leve
~75 MB por agente em idle. 10, 20, 50 agentes na mesma VPS de $5/mês. Zero overhead de container.
Multi-provedor de IA
Claude (API ou CLI), OpenRouter (Grok, GPT-4o, Gemini), OpenAI — tudo com OAuth automático ou API key.
Segurança por padrão
Tokens isolados, sandbox de comandos, controle de acesso por aprovação, credenciais com chmod 600.
Memória em camadas
Personalidade, perfil do usuário, memória de longo prazo e diários diários auto-destilados toda noite.
Ferramentas modulares
Shell, HTTP, Git, GitHub, banco de dados, arquivos, cron — ative só o que cada agente precisa.
Painel web
Dashboard FastAPI com setup wizard, editor de configuração, logs em tempo real e gerenciamento de usuários.
Subagentes
Delegue tarefas para agentes especializados — geração de imagens, análise de dados, qualquer especialidade.
Bug Fixer autônomo
Agente que monitora erros nos logs, chama o Claude para analisar e te notifica no Telegram.
Instalação

Rodando em minutos

Sem Docker, sem Kubernetes, sem drama.

1
Clone o repositório
Python 3.10+ e Linux com systemd são os únicos pré-requisitos obrigatórios.
git clone https://github.com/seu-usuario/smb-claw.git && cd smb-claw
2
Rode o setup
O script verifica dependências, detecta autenticações existentes e abre o painel admin com o wizard de configuração.
bash setup.sh
3
Configure pelo painel web
Acesse http://<ip>:8080 — escolha o provedor de IA, configure o Telegram Token e crie seu primeiro agente com personalidade, ferramentas e modo de acesso.
4
Crie quantos agentes quiser
Cada agente é um serviço systemd independente — isolado, resiliente e com reinício automático.
bash criar-bot.sh meu-agente
