Agentes Telegram que escalam com você.

O framework resiliente para gerenciar múltiplos agentes de IA. Do setup no terminal à operação total via Telegram e Painel Web.

Começar Instalação Documentação
Instalação via Terminal
git clone https://github.com/alozs/smb-claw
cd smb-claw && bash setup.sh

01. Setup Rápido

Um único comando para configurar dependências, detectar provedores (Claude, OpenRouter, Codex) e iniciar o Painel Admin.

02. Gestão via Telegram

Crie agentes e sub-agentes com /criar_agente. Controle tudo via chat, sem precisar tocar no servidor novamente.

03. Painel de Controle

Acesse o painel web para logs em tempo real e ajustes finos. Protegido por tokens temporários gerados no Telegram.

Recursos de Elite

Segurança

Acesso Temporário

O painel administrativo utiliza autenticação por tokens in-memory com validade de 30 minutos. Sem senhas estáticas expostas no servidor.

IA

Sub-agentes

Delegue tarefas complexas para agentes especializados criados dinamicamente via chat.

Memória em Camadas

Sistema híbrido: Soul (regras), USER.md (preferências), MEMORY.md (longo prazo) e histórico SQLite (curto prazo).

Bug Fixer Autônomo

Monitoramento proativo de logs. Detecta falhas e sugere correções ou alerta o admin automaticamente via Telegram.

Analytics

Acompanhe consumo de tokens, latência e custo por agente com o comando /stats.

v0.7.0

Modo Raciocínio (Thinking)

Suporte nativo ao comando /thinking com orçamentos de até 16k tokens de pensamento interno para problemas de alta complexidade.
