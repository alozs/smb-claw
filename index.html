O Ciclo de Vida do Agente

Desenvolvido para ser configurado uma vez e operado de qualquer lugar.

01. Instalação Zero-Drama

Use o terminal apenas para o essencial. O setup configura seu ambiente, detecta o provedor e sobe os serviços de base.

$ bash setup.sh

02. Operação Via Chat

Crie novos agentes, defina personalidades e execute comandos administrativos diretamente pelo Telegram. O terminal agora é opcional.

/criar_agente

03. Gestão Web Segura

Acesse logs em tempo real, gerencie memória de longo prazo e edite configurações pelo painel web protegido por tokens temporários.

✔ Access Token: 4j2k...

Segurança e Governança

Agentes poderosos exigem responsabilidade e isolamento.

ADMIN AUTH

Tokens com Expiração

O painel admin não usa senhas estáticas. Acesse via tokens gerados pelo bot que expiram em 2 horas ou no próximo reinício.

SANDBOX

Shell Denylist

Proteção nativa contra comandos destrutivos. O agente pode rodar scripts, mas nunca deletar o próprio núcleo ou parar serviços críticos.

REASONING

Extended Thinking

Suporte nativo a orçamentos de pensamento (tokens de raciocínio) para tarefas complexas.

Memória Destilada em Camadas

Short-term

Contexto imediato da conversa via banco de dados SQLite.

Long-term

Arquivos .md destilados automaticamente toda noite.

Soul

Instruções fixas que definem a ética e o comportamento.
