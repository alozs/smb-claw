novo: v0.7.0
Modo Pensamento (Extended Reasoning)
Agora com suporte ao comando /thinking. Configure o nível de raciocínio (off, low, medium, high) para resolver problemas complexos com orçamentos de até 16.000 tokens de pensamento interno.
Docker Nativo
Suporte completo a ambientes containerizados com detecção automática. Use Systemd na VPS ou Docker no Cloud — o SMB Claw se adapta.
Memória Destilada
Compactação automática de memória (MEMORY.md) quando excede 6KB. Seus agentes nunca esquecem o que importa, mas não ficam lentos.
Segurança Isolada
Tokens protegidos por lock files, denylist de comandos destrutivos no shell e chmod 600 automático em arquivos sensíveis.
Histórico Infinito
Mensagens que excedem o contexto são arquivadas em banco de dados SQLite, prontas para auditoria e recuperação.
Truncate Inteligente
Saídas de ferramentas muito longas são truncadas em 12k caracteres, evitando erros de contexto sem quebrar a lógica do agente.
