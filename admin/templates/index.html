Bug Fixer Agent

Detecta erros nos agentes via analytics e journalctl, e invoca Claude para corrigi-los automaticamente.

Ativado

Quando ativo, o agente roda automaticamente conforme a frequência configurada.

O token do agente que enviará os alertas ao admin. Copie de qualquer bots/<nome>/.env → TELEGRAM_TOKEN.

Última execução:

Como funciona

  • Verifica o banco de analytics de cada agente em busca de erros desde a última execução
  • Se erros forem encontrados, coleta os logs do journalctl e envia para o Claude corrigir
  • Claude lê os arquivos fonte, aplica a correção mínima, reinicia o agente e reporta o resultado
  • O admin recebe notificação no Telegram com o diagnóstico e a correção aplicada
  • Erros transitórios (timeouts de rede) são identificados e não geram modificações
